# Modern Web Development with React

**Duration:** 10 Weeks 
**Schedule:** 2 Hours per Week 
**Audience:** Beginners with basic JavaScript knowledge 

---

## Course Description

This course introduces modern frontend development using React. 
Students will build a real-world web application step by step while learning core React concepts including components, state management, routing, API integration, and deployment.

The course is highly practical and project-driven. By the end, students will deploy a fully functional React application.

---

## Learning Outcomes

By the end of this course, students will be able to:

- Build reusable React components
- Use JSX effectively
- Manage state using React hooks
- Handle forms and user input
- Work with side effects and local storage
- Implement routing in a multi-page application
- Manage global state with Context API
- Fetch and manage API data
- Optimize React applications
- Deploy a production-ready React app

---

## Course Project Thread

Throughout the course, students will build a progressively enhanced application:

### **Smart Task Manager**

Features added incrementally:
- Task display
- Task creation
- State management
- Persistence with localStorage
- Multi-page routing
- Global state
- API integration
- Deployment

## Week 1 – React Fundamentals & Components

**Topics**
- What is React?
- JSX
- Functional components
- Props

## Week 2 – State & Interactivity

- Event handling
- Rendering lists

- Toggle completed tasks

## Week 3 – Forms & Controlled Components

**Topics**
- Controlled inputs
- Form submission
- Basic validation

**Hands-On**
- Build Add Task form
- Validate required fields

**Outcome**
Functional task creation workflow

## Week 4 – Side Effects & Persistence

**Topics**
